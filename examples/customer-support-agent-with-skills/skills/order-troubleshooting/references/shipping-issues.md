# Shipping Issues

## Delayed shipment diagnosis

1. Check if the order status is `shipped` and the estimated delivery date has passed.
2. Look up carrier tracking via the tracking number in the order.
3. Common causes:
   - Carrier delays (weather, volume)
   - Customs hold (international orders)
   - Incorrect or incomplete address

## Resolution

- **Less than 7 days past estimated delivery**: advise the customer to wait and monitor tracking. Carrier delays are common and usually resolve within a few days.
- **7+ days past estimated delivery**: offer to file a carrier claim and either re-ship the order or issue a full refund (customer's choice).
- **Incorrect address**: if the package is still in transit, attempt a carrier intercept. If delivered to the wrong address, re-ship at no cost.

## International orders

- Customs holds can add 5-15 business days. Check if a customs fee is pending.
- If the package is returned to sender by customs, re-ship with corrected documentation or issue a refund.
